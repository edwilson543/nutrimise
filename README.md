### Web app for optimising meal plans

---

### System requirements:
- Python 3.11
- postgres 15.1
- mkcert 1.4.4 (optional)

---

### Installation:

```bash
python3.11 -m venv venv
source venv/bin/activate
make install
```

---

### Run the development server:
With HTTP:
```bash
make server
```

With HTTPS:
```bash
make server_https
```
